## Requirements

- python = 3.12+
- networkx = 3.3+
- numpy = 2.1.1+
- PyQt5 = 5.15.11+
- pyvis = 0.3.2+
- scipy = 1.14.1+
- matplotlib = 3.9.2+

## Installation

Clone the repository:

```bash
git clone https://github.com/Ivan-Pokhabov/GraphResearh_ForCheckingCriterionOfGoodnessOfFit
```

Install dependencies:

```bash
poetry install
```

## Development

Install requirements

```bash
poetry install --with dev
```
